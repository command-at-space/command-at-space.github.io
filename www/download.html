INSTALL

Updated: May 20th 2020

Version: 0.0.0

Windows

i) Download win64.zip and unzip. It will create a win64 folder

ii) Enter win64 folder and double-click CaS.exe

Linux

i) Download linux64.zip and unzip. It will create a linux64 folder

ii) Enter linux64 folder and execute start.sh
cd linux64
./start.sh

Controls

i) ESC - Close Game
